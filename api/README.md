# Documentation API

This package facilitates all the services required by `documentation` Web UI.

It is a set of services running on NodeJS connecting to AWS DynamoDB.

## Pre-requisites

```shell
npm install
```

## Development

This will run the Web UI at <http://localhost:4000>

```shell
npm run start-dev
```
